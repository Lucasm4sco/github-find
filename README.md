# GitHub Finder

## Sobre o projeto

Aplicação desenvolvida com TypeScript em React que nos retorna dados do GitHub sobre o usuário procurado.

## [App Link](https://github-find-nu.vercel.app/)

Deploy realizado no Vercel.
